# Earn From Estate 
Real estate renting platform, that connects the two ends of the market.

A user would have opportunity to register as a tenant or as landlord, and
based on the account type he will be given different tools to work with.

## Getting Started
Checkout the git repository, no need of additional setup so far, cause the 
project uses embedded database.

### Prerequisites 
What things you need in order to install & run the software:
* JDK 1.8 + 

#### Technologies
The technologies that are used and will be used in the project are as follows:

Backend
* REST architecture
* Maven
* Spring Boot
* Spring JPA
* Spring JDBC
* Hibernate Core
* H2 embedded DB - will change to some other relational db.

Frontend
* ReactJS
